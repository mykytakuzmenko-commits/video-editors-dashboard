# Video editors niche — market dashboard

Single-file interactive dashboard (revenue & downloads, 2023–2026 projection) for the video-editors app niche, with a With / W.o. CapCut toggle.

## View
- Open `index.html` (same as `video-editors-dashboard.html`).

## Deploy on Vercel
Static site, zero config — import the repo in Vercel and deploy. `index.html` is served at the root URL.
